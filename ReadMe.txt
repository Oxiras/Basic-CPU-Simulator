Included:
- Memory.java: Memory program of the application.
- Processor.java: Processor program of the application.
- sample5.txt: Sample created for sixth output. Note that it uses a different approach to printing the date
- Prject 1 Summary: Summary of the project.

Follow these steps to run the application:
1. Place following files in the same folder:
	- Memory.java
	- Processor.java
	- All sample files
2. Run following commands:
	- javac Memory.java Processor.java
	- java Processor sampleName timerFlag 
		(timerFlag as an int to indicate the number of instructions until timer interrupt is triggered)
		Ex: java Processor sample2.txt 30
